Vive de Lujo

Transformación digital para la gestión y promoción de inmuebles en Valledupar, Colombia.

¿Qué ofrece Vive de Lujo?

Vive de Lujo es una aplicación móvil desarrollada con el objetivo de gestionar y promocionar inmuebles en la ciudad de Valledupar. Esta plataforma busca optimizar la interacción entre la inmobiliaria y sus clientes, ofreciendo una experiencia ágil, visual e intuitiva para la búsqueda, consulta y exploración de propiedades disponibles, mejorando así el proceso de compra o alquiler de bienes raíces.

Tecnologías Utilizadas

Dart: Lenguaje principal para desarrollar la lógica de la aplicación en Flutter.

Flutter: Framework para crear la interfaz móvil con un diseño moderno y responsivo.

Git: Control de versiones para llevar un seguimiento del desarrollo.

Firebase: Backend en la nube para autenticación, base de datos y almacenamiento.

Test: Pruebas automatizadas para garantizar el correcto funcionamiento de la app.
