Plataforma Web del Consultorio Jurídico

Gestione casos, citas y seguimiento jurídico de manera eficiente.

¿Qué ofrece la plataforma?

Aplicativo web diseñado para el Consultorio Jurídico de la Universidad Popular del Cesar. Permite a los ciudadanos registrar solicitudes legales, agendar citas y recibir asesoría. Mejora la organización interna, agiliza la atención, mantiene un historial de casos y garantiza trazabilidad. Optimiza los procesos, reduce tiempos y fortalece el acceso a la justicia para personas de bajos recursos.

Tecnologías Utilizadas

Angular: Para construir la interfaz de usuario dinámica del sistema.

HTML: Estructura básica del sitio web.

CSS: Diseño visual responsivo y estilizado.

TypeScript: Lógica robusta del frontend con Angular.

Node.js: Backend para manejar peticiones y lógica del servidor.

MySQL: Base de datos para almacenamiento de la información legal.
